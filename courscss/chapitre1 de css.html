Chapitre 1 : Introduction au CSS

1. Qu'est-ce que le CSS ?

Le CSS (Cascading Style Sheets) est un langage qui permet de mettre en forme une page web HTML. Il sert à définir les couleurs, les polices, les tailles, les espacements, etc.

2. Historique

Le CSS a été inventé par Håkon Wium Lie en 1996 alors qu'il travaillait au W3C (World Wide Web Consortium).

3. Syntaxe de base

La syntaxe CSS est simple :

    sélecteur {
      propriété: valeur;
    }

Exemple :

    p {
      color: red;
    }

4. Les trois méthodes d'intégration CSS

  • En ligne : dans une balise HTML avec l'attribut style
  • Interne : dans la balise <style> du fichier HTML
  • Externe : dans un fichier .css séparé lié via <link>
    <style>
      h1 { color: blue; }
    </style>

5. Exemple simple

    <!DOCTYPE html>
    <html>
    <head>
      <style>
        p {
          color: green;
          font-size: 18px;
        }
      </style>
    </head>
    <body>
      <p>Ceci est un paragraphe stylisé avec CSS.</p>
    </body>
    </html>
