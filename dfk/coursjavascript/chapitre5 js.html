Chapitre 5 : Fonctions en JavaScript

Introduction

Les fonctions permettent d'organiser du code en blocs réutilisables. Une fonction est un groupe d'instructions exécutées lorsqu'on l'appelle.

Déclaration d'une fonction

function saluer() {
  alert("Bonjour !");
}

saluer(); // Appel de la fonction

Fonction avec paramètres

function direBonjour(nom) {
  console.log("Bonjour " + nom);
}

direBonjour("Fatou");

Fonction avec valeur de retour

function addition(a, b) {
  return a + b;
}

let resultat = addition(4, 7);
console.log(resultat);

Fonction anonyme et fléchée

// Fonction anonyme
let direHello = function() {
  console.log("Hello");
};

direHello();

// Fonction fléchée (ES6)
let carre = (x) => x * x;
console.log(carre(5));

Devoir

  1. Créer une fonction multiplier qui prend deux nombres et retourne le produit.
  2. Créer une fonction estPair qui vérifie si un nombre est pair.
  3. Créer une fonction fléchée qui calcule le carré d’un nombre.
  4. Créer une fonction qui affiche "Bienvenue [nom]" avec le nom passé en paramètre.
  5. Créer une fonction qui calcule la moyenne de 3 notes.
