Chapitre 5 : Formulaires HTML de base et avancés

1. Qu'est-ce qu'un formulaire ?

Un formulaire HTML permet de collecter des données auprès des utilisateurs, comme du texte, des choix multiples, des dates, etc. Ces données peuvent ensuite être envoyées vers un serveur pour traitement.

2. Les balises principales d’un formulaire

  • <form> : conteneur du formulaire, avec des attributs comme action (URL du serveur) et method (GET ou POST).
  • <input> : champ de saisie (texte, email, mot de passe, bouton radio, checkbox, etc.).
  • <textarea> : zone de texte multilignes.
  • <select> et <option> : menu déroulant.
  • <button> : bouton pour soumettre ou autre action.
  • <label> : permet d’associer un texte explicatif à un champ.

3. Exemple simple de formulaire

<form action="/submit" method="post">
  <label for="nom">Nom :</label>
  <input type="text" id="nom" name="nom" required />

  <label for="email">Email :</label>
  <input type="email" id="email" name="email" required />

  <button type="submit">Envoyer</button>
</form>

4. Types d’input courants

  • text : champ texte simple
  • email : champ email avec validation basique
  • password : champ mot de passe masqué
  • radio : choix unique parmi plusieurs options
  • checkbox : choix multiple
  • submit : bouton pour envoyer le formulaire
  • reset : bouton pour réinitialiser les champs
  • date, number, color etc. : champs spécialisés
