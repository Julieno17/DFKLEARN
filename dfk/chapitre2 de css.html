Chapitre 2 : Les Sélecteurs CSS

Les sélecteurs CSS permettent de cibler les éléments HTML à styliser.

1. Sélecteur de type

Sélectionne tous les éléments d'un type donné :

p { color: red; }

2. Sélecteur de classe

Cible les éléments avec une classe spécifique :

.important { font-weight: bold; }

3. Sélecteur d'identifiant (ID)

Cible un élément unique avec un ID :

#titre { text-align: center; }

4. Sélecteurs combinés

Vous pouvez combiner les sélecteurs :

div p.important { color: blue; }

Exemple

<style>
  p { color: green; }
  .rouge { color: red; }
  #titre { font-size: 24px; }
</style>

<h1 id="titre">Mon titre</h1>
<p>Paragraphe normal</p>
<p class="rouge">Paragraphe rouge</p>
