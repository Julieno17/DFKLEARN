Chapitre 4 : Mise en page et positionnement en CSS

Introduction

En CSS, la mise en page est essentielle pour organiser le contenu d’une page web. Ce chapitre présente les concepts clés pour contrôler la disposition des éléments : le modèle de boîte, le positionnement, et les systèmes modernes comme Flexbox et Grid.

1. Le modèle de boîte CSS (Box Model)

Chaque élément HTML est une boîte composée de :

  • Contenu : texte ou images à l’intérieur
  • Padding (remplissage) : espace entre le contenu et la bordure
  • Border (bordure) : contour de la boîte
  • Margin (marge) : espace entre cette boîte et les autres éléments

Exemple visuel :

+----------------------------+
|          Margin            |
|  +----------------------+  |
|  |      Border          |  |
|  |  +----------------+  |  |
|  |  |    Padding     |  |  |
|  |  |  +----------+  |  |  |
|  |  |  | Contenu  |  |  |  |
|  |  |  +----------+  |  |  |
|  |  +----------------+  |  |
|  +----------------------+  |
+----------------------------+

Propriétés CSS liées :

div {
  margin: 20px;            /* marge extérieure */
  padding: 10px;           /* espace intérieur */
  border: 2px solid black; /* bordure */
  width: 200px;            /* largeur du contenu */
}

2. Propriété box-sizing

Par défaut, la largeur et la hauteur concernent seulement le contenu, pas le padding ni la bordure. La propriété box-sizing: border-box; inclut padding et bordure dans la taille.

div {
  box-sizing: border-box;
}

3. Propriété display

Contrôle le type d’affichage d’un élément :

  • block : élément en bloc, prend toute la largeur, commence sur une nouvelle ligne
  • inline : élément en ligne, ne commence pas une nouvelle ligne
  • inline-block : mixte, comme inline mais on peut fixer taille
  • none : élément caché
p {
  display: block;
}
span {
  display: inline;
}
button {
  display: inline-block;
}
.hide {
  display: none;
}

4. Positionnement des éléments

4.1 Position statique (par défaut)

Position normale dans le flux.

div {
  position: static;
}

4.2 Position relative

Décalage par rapport à la position normale, mais espace réservé.

div {
  position: relative;
  top: 10px;
  left: 20px;
}

4.3 Position absolute

Positionné par rapport au premier ancêtre positionné, hors flux.

div {
  position: absolute;
  top: 50px;
  left: 100px;
}

4.4 Position fixed

Fixé par rapport à la fenêtre, ne bouge pas au scroll.

div {
  position: fixed;
  top: 0;
  right: 0;
}

4.5 Position sticky

Comme relative jusqu’à atteindre un seuil, puis fixé.

header {
  position: sticky;
  top: 0;
  background: white;
}

5. Flexbox : Mise en page flexible

Permet d’organiser les éléments en ligne ou colonne facilement.

.container {
  display: flex;
  flex-direction: row; /* ou column */
  justify-content: center; /* alignement horizontal */
  align-items: center; /* alignement vertical */
  gap: 10px; /* espace entre éléments */
}

Enfants flex :

.item {
  flex: 1; /* chaque enfant prend la même place */
}

6. Grid Layout : mise en page en grille

Crée facilement une grille en lignes et colonnes.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 colonnes égales */
  gap: 20px;
}

7. Float et clear (ancienne méthode)

float fait flotter un élément à gauche ou droite, clear empêche le chevauchement.

img {
  float: left;
  margin-right: 10px;
}
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
