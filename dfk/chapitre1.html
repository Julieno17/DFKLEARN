Chapitre 1 : Introduction au HTML

Qu’est-ce que le HTML ?

HTML signifie HyperText Markup Language, soit en français langage de balisage hypertexte. C’est le langage standard utilisé pour créer des pages web.

Le HTML sert à structurer le contenu d’une page : textes, images, liens, tableaux, formulaires, etc.

Ce langage a été inventé par Tim Berners-Lee en 1991. Depuis, il a beaucoup évolué pour devenir la base du web moderne.

Le rôle de la déclaration <!DOCTYPE html>

La première ligne <!DOCTYPE html> est une déclaration spéciale qui indique au navigateur que la page utilise la norme HTML5. Cela permet au navigateur d’interpréter correctement le code.

Note : Ce n’est pas une balise HTML, mais une instruction pour le navigateur.

Structure de base d’une page HTML

Voici un exemple minimaliste d’une page HTML complète :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma première page</title>
</head>
<body>
    <h1>Bonjour le monde !</h1>
    <p>Bienvenue dans votre premier document HTML.</p>
</body>
</html>

Explications :

  • <!DOCTYPE html> : déclaration du type de document (HTML5).
  • <html lang="fr"> : racine du document, avec langue française.
  • <head> : partie "en-tête" où on place les métadonnées, le titre, etc.
  • <meta charset="UTF-8"> : définit l’encodage des caractères (important pour afficher les accents).
  • <title> : titre de la page affiché dans l’onglet du navigateur.
  • <body> : contenu visible de la page.
  • <h1> : titre principal visible sur la page.
  • <p> : paragraphe de texte.

Les balises orphelines (auto-fermantes)

Certaines balises HTML n’ont pas besoin de balise fermante, elles sont dites orphelines ou auto-fermantes. On les utilise pour insérer des éléments qui ne contiennent pas de contenu, comme une image, un saut de ligne, ou une ligne horizontale.

Exemples courants :

<br />  <!-- saut de ligne -->
<hr />  <!-- ligne horizontale -->
<img src="image.jpg" alt="Description" />  <!-- image -->

En HTML5, il est permis d’écrire ces balises sans la barre oblique finale, comme <br> ou <img src="..." alt="...">, et le navigateur comprendra quand même.

Important : Contrairement à la majorité des balises, ces balises n’ont pas de contenu à l’intérieur, donc pas de balise fermante.

Les balises non orphelines (avec contenu)

Contrairement aux balises orphelines, la plupart des balises HTML sont non orphelines. Elles contiennent du contenu et doivent être ouvertes et fermées.

Exemple :

<h1>Titre principal</h1>
<p>Ceci est un paragraphe.</p>
<div>
    <p>Un paragraphe dans une division.</p>
</div>

On voit ici que chaque balise d’ouverture comme <p> doit être fermée par </p>.

Exercice pratique

Créez un fichier nommé index.html avec le contenu donné plus haut, puis ouvrez-le dans votre navigateur web pour voir le résultat.

Ajoutez dans le <body> un saut de ligne avec <br>, une image avec <img> (utilisez une image sur votre disque ou une URL) et une ligne horizontale avec <hr>.

Exemple :

<body>
    <h1>Bonjour le monde !</h1>
    <p>Bienvenue dans votre premier document HTML.</p>
    <br>
    <img src="https://via.placeholder.com/150" alt="Image exemple">
    <hr>
</body>
