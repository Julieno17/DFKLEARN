Chapitre 5 : Animation et transitions en CSS

Introduction

Les animations et transitions CSS permettent d'ajouter des effets visuels fluides et dynamiques aux éléments d’une page web, améliorant ainsi l’expérience utilisateur.

1. Transitions CSS

Les transitions permettent d’animer en douceur le changement d’une propriété CSS lorsqu’un événement se produit (comme le passage de la souris).

.element {
  transition: propriété durée fonction_timing délai;
}

Exemple :

.box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition: background-color 0.5s ease;
}

.box:hover {
  background-color: red;
}

2. Propriétés animables avec transitions

Exemples : color, background-color, width, height, opacity, transform…

3. Animation CSS avec @keyframes

Permet de créer une animation plus complexe en définissant plusieurs étapes.

@keyframes monAnimation {
  0%   { transform: translateX(0); }
  50%  { transform: translateX(100px); }
  100% { transform: translateX(0); }
}

.element {
  animation-name: monAnimation;
  animation-duration: 2s;
  animation-iteration-count: infinite;
  animation-timing-function: linear;
}

4. Propriétés d’animation importantes

  • animation-name : nom de l’animation définie par @keyframes
  • animation-duration : durée (ex : 2s)
  • animation-iteration-count : nombre de répétitions (ex : infinite)
  • animation-timing-function : fonction de vitesse (linear, ease, ease-in, ease-out)
  • animation-delay : délai avant début de l’animation
