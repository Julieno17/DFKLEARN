Chapitre 4 : Tableaux et Listes en HTML

Les tableaux en HTML

Les tableaux permettent d’organiser des données sous forme de lignes et colonnes. Ils sont très utiles pour afficher des informations structurées comme des horaires, des tarifs, des listes comparatives, etc.

1. Les balises principales d'un tableau

  • <table> : définit le tableau
  • <tr> (table row) : définit une ligne du tableau
  • <th> (table header) : définit une cellule d’en-tête (titre de colonne ou ligne)
  • <td> (table data) : définit une cellule de données

2. Exemple simple de tableau

<table>
  <tr>
    <th>Nom</th>
    <th>Âge</th>
    <th>Ville</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
    <td>Paris</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
    <td>Lyon</td>
  </tr>
</table>

Ce code crée un tableau avec 3 colonnes et 3 lignes (dont la ligne d’en-tête).

3. Exemple affiché

Nom Âge Ville
Alice 25 Paris
Bob 30 Lyon

4. Attributs utiles

  • border="1" : ajoute une bordure (ancien usage, on recommande CSS)
  • colspan="x" : fusionne x colonnes dans une cellule
  • rowspan="x" : fusionne x lignes dans une cellule

5. Exemple avec colspan et rowspan

<table border="1">
  <tr>
    <th>Produit</th>
    <th colspan="2">Prix</th>
  </tr>
  <tr>
    <td rowspan="2">Pomme</td>
    <td>Au kilo</td>
    <td>2€</td>
  </tr>
  <tr>
    <td>À la pièce</td>
    <td>0,50€</td>
  </tr>
</table>

6. Résultat affiché

Produit Prix
Pomme Au kilo 2€
À la pièce 0,50€

Les listes en HTML

Les listes permettent d’organiser des éléments sous forme ordonnée ou non ordonnée.

1. Liste non ordonnée (puces)

On utilise la balise <ul> pour la liste non ordonnée, et chaque élément est défini avec <li>.

<ul>
  <li>Pommes</li>
  <li>Bananes</li>
  <li>Cerises</li>
</ul>

2. Liste ordonnée (numérotée)

On utilise la balise <ol> pour la liste ordonnée.

<ol>
  <li>Préparer la pâte</li>
  <li>Faire cuire au four</li>
  <li>Laisser refroidir</li>
</ol>

3. Listes imbriquées

On peut imbriquer une liste dans une autre pour des sous-catégories :

<ul>
  <li>Fruits
    <ul>
      <li>Pommes</li>
      <li>Bananes</li>
    </ul>
  </li>
  <li>Légumes</li>
</ul>

4. Exemple affiché

  • Fruits
    • Pommes
    • Bananes
  • Légumes
