Chapitre 3 : Couleurs, Polices et Boîtes en CSS

1. Les couleurs

On peut définir les couleurs avec des noms, des codes hexadécimaux (#RRGGBB), ou RGB().

p {
  color: red;
  background-color: #f2f2f2;
}

2. Les polices

Les polices sont définies avec la propriété font-family. D'autres propriétés utiles sont : font-size, font-weight et font-style.

h1 {
  font-family: 'Arial', sans-serif;
  font-size: 24px;
  font-weight: bold;
}

3. Le modèle de boîte (Box Model)

Chaque élément HTML peut être considéré comme une boîte composée de :

  • Contenu (content)
  • Remplissage (padding)
  • Bordure (border)
  • Marge (margin)
div {
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}
