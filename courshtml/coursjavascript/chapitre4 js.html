Chapitre 4 : Les Tableaux en JavaScript

Introduction

Un tableau en JavaScript est une structure de données utilisée pour stocker plusieurs valeurs dans une seule variable.

Création d'un tableau

let fruits = ["Pomme", "Banane", "Orange"];

On peut aussi créer un tableau vide puis y ajouter des éléments :

let nombres = [];
nombres[0] = 10;
nombres[1] = 20;

Accès aux éléments

On accède aux éléments du tableau à l'aide de leur index :

let premierFruit = fruits[0]; // "Pomme"

Modifier un élément

fruits[1] = "Mangue";

Longueur du tableau

let longueur = fruits.length;

Itération à travers un tableau

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
